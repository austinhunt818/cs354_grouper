Team Program: Ruby On Rails

For our team program we decided tackle the problem of creating a simple todo list app using Ruby on Rails. This app should allow users to Create, Delete and Update tasks that they want to keep track of. We chose is problem because Ruby's claim to fame and is main use today is in Ruby on Rails. Also because a simple todo list captures the functionality that larger more complex sites require and will allow us to see how easy it is to get things up and running with Ruby on Rails
