Translators and Installation Instructions

Installing ruby onto your system is quite easy, but depending on the system varies. Some of the most common systems will have step by step instructions on how to install.

Debian or Ubuntu

$ sudo apt-get install ruby-full

CentOS, Fedora, or RHEL

$ sudo yum install ruby

snap (Ubuntu or other Linux distrubtions

$ sudo snap install ruby --classic

Checking to see if you already have ruby installed

You can check inside a terminal emulator by typing: $ ruby -v
