Running a Ruby Program

Once ruby is install open a terminal or command prompt, navigate to the directory containing your .rb file, and use the following command


 ruby NameOfYourFile.rb

This will execute the program, and you should see the output:

Example 1. Bubble Sort

          
  def bubble_sort(array)
      n = array.length
      loop do
        swapped = false
        # Iterate from 0 to n-2
        (1...n).each do |i|
          # Compare the current element with the previous one
          if array[i - 1] > array[i]
            # Swap the elements
            array[i - 1], array[i] = array[i], array[i - 1]
            swapped = true
          end
        end
        # Break the loop if no elements were swapped
        break unless swapped
      end
      # Return sorted array
      array
    end
    # Example usage:
    arr = [5, 3, 8, 4, 2]
    puts "Original array: #{arr}"
    sorted_arr = bubble_sort(arr)
    puts "Sorted array: #{sorted_arr}"

Features Highlighted:

  • Iterators and Ranges: The use of (1...n).each demonstrates Ruby's range and iterator functionality,
  • Dynamic Typing: Ruby dynamically determines the types of variables (like array and swapped), eliminating the need for explicit type declarations
  • Readable and Concise Syntax: Ruby's clear and syntax makes algorithms like Bubble Sort easy to implement and understand.

Example 2. Classes

          
  # Base class representing a general greeter
  class Greeter
      # Constructor method: Called when a new instance of the class is created
      # Initializes the greeter with a name
      def initialize(name)
        @name = name  # Instance variable to store the name
      end
    
      # Method to greet using the name
      def greet
        puts "Hello, #{@name}!"  # Output a general greeting message
      end
    end
    
    # Subclass of Greeter specifically for Walmart greeters
    class WalmartGreeter < Greeter
      # Additional method for Walmart-specific greeting
      def store_greet
        # Output a store-specific greeting message
        puts "Hello, #{@name}, welcome to Walmart!"
      end
    end
    
    # Creating an instance of the Greeter class
    greeter = Greeter.new("Ruby")
    greeter.greet  
    
    # Creating an instance of the WalmartGreeter subclass
    walmart_greeter = WalmartGreeter.new("Matz")  
    walmart_greeter.greet 
    walmart_greeter.store_greet

Features Highlighted:

  • Object-Oriented Design: Ruby's classes and inheritance make it easy to create reusable and specialized components, as seen with Greeter and WalmartGreeter.
  • Method Overriding and Extension: The WalmartGreeter subclass extends the Greeter class and adds a new method (store_greet), demonstrating inheritance and specialization
  • Instance Variables: The use of @name showcases Ruby's instance variables, which store data specific to an object

Example 3. Word Frequency Counter


  puts "Enter a sentence:"
  sentence = gets.chomp

  # Split the sentence into words, normalize to lowercase, and count frequencies
  word_counts = Hash.new(0)  # Default value for missing keys is 0
  sentence.split.each { |word| word_counts[word.downcase] += 1 }

  # Sort and display the word counts
  puts "\nWord Frequency:"
  word_counts.sort_by { |_word, count| -count }.each do |word, count|
    puts "#{word}: #{count}"
  end

Features Highlighted:

  • Concise Syntax: Ruby's Hash.new(0) makes it easy to create a hash with default values.
  • Iterators and Blocks: The use of each and sort_by with blocks ({}) showcases Ruby's emphasis on clean, readable code.
  • Dynamic Typing: No need to declare variable types
  • Interactive I/O: The gets method to take user input and puts for output make it beginner-friendly
