Ruby

Ruby is simple in appearance, but is very complex inside, just like our human body.

- Yukihiro “Matz” Matsumoto

General Description

Ruby is an interpreted, high-level, general purpose programming language known for its simplicity and flexibility. Designed by Yukihiro Matsumoto and released in the mid-1990s, Ruby places an emphasis on being user-friendly with a simple syntax that prioritizes developer happiness, productivity and readability. It is dynamically typed, uses garbage-collecting, and just-in-time compilation supporting object-oriented, functional, and imperative programming styles. Ruby is widely recognized for its use in the web development framework Ruby on Rails, which simplifies building full stack web applications. Ruby on Rails has been used to build big names like Shopify, Twitter, Github, Twitch and more. With a strong focus on readability and expressiveness, Ruby allows developers to write concise, yet powerful code, making it a popular choice for both beginners and experienced programmers alike.

History

Ruby was born on February 24th, 1993, as that was the day that creator Yukihiro Matsumoto, otherwise referred to as Matz, picked the name Ruby. Matsumoto wanted to design a language that combined aspects of his favorite languages, including Perl, Smalltalk, Eiffel and Ada. Ruby 0.95 was the first public release, coming out in 1995. For about a decade, the language was only really popular in Japan. That changed in 2004, when David Heinemeier Hansson introduced Ruby on Rails, a web development framework. Rails became very popular for web development throughout the 2000's, leveraging the expressiveness of Ruby to make web development more intuitive for developers. By 2016, ruby was the 8th most popular programming language.

Currently, Ruby's popularity is so-so. With the ever evolving world of web development, Rails became less popular for new web apps, and so therefore did Ruby. However, Ruby is still relevant. Rails may be less popular than more recent frameworks, but it is still used widely, and is still one of the highest paying web technologies. Ruby is still used for its expressiveness and readability, as programmers enjoy being able to write concise and expressive programs. Companies such as Airbnb, Shopify, and Amazon use Rails for some of their websites. While Ruby's popularity may have dwindled a bit in the past decade, the community is still alive and well, and it's clear that Ruby isn't going anywhere anytime soon.

Example 1. Bubble Sort

            
  def bubble_sort(array)
      n = array.length
      loop do
        swapped = false
        # Iterate from 0 to n-2
        (1...n).each do |i|
          # Compare the current element with the previous one
          if array[i - 1] > array[i]
            # Swap the elements
            array[i - 1], array[i] = array[i], array[i - 1]
            swapped = true
          end
        end
        # Break the loop if no elements were swapped
        break unless swapped
      end
      # Return sorted array
      array
    end
    # Example usage:
    arr = [5, 3, 8, 4, 2]
    puts "Original array: #{arr}"
    sorted_arr = bubble_sort(arr)
    puts "Sorted array: #{sorted_arr}"
